# How to use the repository
---------------------------

First clone the repository into the desired folder:

```
$ git clone https://github.com/yacinebouaouni/ACGAN-Xray-Generation-Tensorflow.git
```

Create a new environment and install the requirements

```
$ conda create --name acgan --python=3.9
$ conda install pip 
$ conda activate acgan
$ pip install -r requirements.txt
$ pip install -e .
```
